## 二、脚本结构（极简三要素）

```
╔═══════════════════════════════════════════════════════╗
║                  我的第一条脚本                        ║
╠═══════════════════════════════════════════════════════╣
║  当 [温度寄存器] [大于] [30] 的时候                  ║
║       ↓                                            ║
║  就 [写入] [关闭阀门] 到 [阀门寄存器]                ║
╚═══════════════════════════════════════════════════════╝
```

### 三要素

| 要素 | 说明 | 示例 |
|------|------|------|
| **触发条件** | 什么时候触发？ | 温度 > 30 |
| **动作** | 触发后做什么？ | 关闭阀门 |
| **目标** | 操作哪个寄存器？ | 阀门寄存器 |

## 三、支持的触发条件

### 3.1 数值比较

| 条件 | 符号 | 示例 |
|------|------|------|
| 大于 | > | 温度 > 30 |
| 小于 | < | 压力 < 100 |
| 等于 | = | 状态 = 1 |
| 不等于 | ≠ | 状态 ≠ 0 |
| 大于等于 | ≥ | 液位 ≥ 80 |
| 小于等于 | ≤ | 液位 ≤ 20 |

### 3.2 变化检测

| 条件 | 说明 | 示例 |
|------|------|------|
| 变成ON | 从0变成1 | 开关变成ON |
| 变成OFF | 从1变成0 | 开关变成OFF |
| 发生变化 | 任意变化 | 任意数值变化 |
| 稳定N秒 | 值不变N秒 | 保持10秒 |

### 3.3 组合条件（简单版）

> 简化版：只支持 AND，不支持嵌套

```
当 [温度 > 30] 且 [压力 < 100] 的时候
```

## 七、技术实现方案

### 7.1 数据结构

```rust
// 脚本定义
struct Script {
    id: Uuid,
    name: String,
    trigger: Trigger,
    actions: Vec<Action>,
    enabled: bool,
}

// 触发条件
enum Trigger {
    // 数值比较
    Compare {
        register: RegisterRef,
        operator: CompareOp,  // >, <, =, ≠, ≥, ≤
        value: f64,
    },
    // 变化检测
    Changed(RegisterRef),
    BecameOn(RegisterRef),
    BecameOff(RegisterRef),
}

// 执行动作
enum Action {
    WriteValue { register: RegisterRef, value: u16 },
    WriteOn(RegisterRef),
    WriteOff(RegisterRef),
    Toggle(RegisterRef),
    ShowNotification(String),
    PlaySound(SoundType),
    Log(String),
    RunScript(Uuid),
    StopScript(Uuid),
}
```

### 7.2 执行引擎

```rust
// 脚本执行器
struct ScriptEngine {
    scripts: HashMap<Uuid, Script>,
    running_scripts: HashSet<Uuid>,
    last_values: HashMap<RegisterRef, u16>,
}

impl ScriptEngine {
    // 每个扫描周期调用
    pub fn evaluate(&mut self, registers: &HashMap<RegisterRef, u16>) {
        for script in self.scripts.values() {
            if !script.enabled || self.running_scripts.contains(&script.id) {
                continue;
            }
            
            // 检查触发条件
            if self.check_trigger(&script.trigger, registers) {
                // 执行动作
                self.execute_actions(&script.actions).await;
            }
        }
    }
}
```
